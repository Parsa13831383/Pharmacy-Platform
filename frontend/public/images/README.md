# Brand Photography

Place the following image files in this directory.
All images are used via Next.js Image component with `fill` layout.

## Required files

| File | Used in | Recommended size |
|---|---|---|
| `hero.jpg` | Hero section (right column, 50% viewport) | 1400 × 1800px (portrait) |
| `brand-story.jpg` | Brand story section | 1200 × 900px (landscape) |
| `promo-banner.jpg` | Dark promotional banner | 1200 × 800px (landscape) |
| `cat-skincare.jpg` | Skincare category card | 800 × 1100px (portrait) |
| `cat-hair-care.jpg` | Hair care category card | 800 × 1100px (portrait) |
| `cat-cosmetics.jpg` | Cosmetics / beauty category card | 800 × 1100px (portrait) |
| `cat-supplements.jpg` | Supplements category card | 800 × 1100px (portrait) |
| `cat-hygiene.jpg` | Personal hygiene category card | 800 × 1100px (portrait) |

## Style guide

- Warm beige / cream backgrounds
- Natural or studio lighting
- Editorial / luxury wellness aesthetic
- Minimal props, clean surfaces
- No text in the photography (text is added in code)
- JPEG at 80–85% quality for web
